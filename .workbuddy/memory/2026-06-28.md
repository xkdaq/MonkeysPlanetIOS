# 2026-06-28

## iOS 版猴哥星球 Flutter 项目实现

参考 Android 项目 `MonkeysPlanetAndroid`，为 iOS 项目 `MonkeysPlanetIOS`（Flutter 空白项目）实现了两个核心功能模块：刷题 和 我的。

### 完成的工作

1. **项目搭建**
   - 更新 pubspec.yaml，添加 12 个依赖（dio, provider, shared_preferences, flutter_secure_storage, webview_flutter, cached_network_image, intl, crypto, pointycastle, url_launcher 等）
   - 创建完整的 lib/ 目录结构（constants, models, services, providers, pages, widgets）

2. **网络层**（完全参考 Android）
   - `HttpClient` - 基于 Dio，集成 SignInterceptor（MD5 签名）、TokenInterceptor（Bearer Token）、ExamDecryptInterceptor（AES/CBC/PKCS5 解密，使用 pointycastle 库）
   - `AuthStorage` - Token 使用 flutter_secure_storage 安全存储，用户信息使用 shared_preferences
   - `CryptoService` - 使用 pointycastle 实现 AES/CBC/PKCS5 解密
   - `ExamService` + `UserService` - 完整 API 调用封装

3. **数据模型**（与 Android 一致）
   - `ApiResponse<T>` - 通用响应包装（兼容 data/rows 双字段）
   - `UserInfo`, `Bank`, `Category`, `Question`, `StartPracticeResult`, `PracticeRecord`, `AnswerRecord`

4. **状态管理**
   - `AuthProvider` - 登录/注册/退出/用户信息
   - `ExamProvider` - 题库列表/分类树/练习/收藏/错题/学习记录

5. **UI 页面**（参考 Android 布局）
   - **MainTabs** - 底部导航栏（刷题、我的 2 个 Tab）
   - **ExamListPage** - 题库列表（卡片式布局）
   - **BankDetailPage** - 题库详情（4种练习模式入口 + 分类树）
   - **PracticePage** - 核心刷题页（答题/背题模式切换、选项渲染、多选/单选/判断、解析展示、答题卡、收藏、计时）
   - **QuestionListPage** - 错题/收藏列表
   - **ProfilePage** - 我的（登录/未登录双状态、头像、菜单列表、退出登录）
   - **LoginPage** - 手机号+密码登录
   - **RegisterPage** - 手机号+验证码+密码注册（60s 倒计时、密码可见性切换）
   - **EditProfilePage** - 编辑资料（昵称、性别）
   - **BindPhonePage** - 绑定手机号
   - **ChangePasswordPage** - 修改密码
   - **StudyRecordsPage** - 学习记录（统计 + 列表）
   - **AboutPage** - 关于我们
   - **FeedbackPage** - 问题反馈
