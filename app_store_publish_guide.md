**是否必填：** 是

**限制：** 最多 4000 个字符

**建议填写：**
```
猴哥星球是一款专为考研学子打造的个人学习管理工具，致力于帮助考生高效备考、巩固知识，让每一次学习都更有价值。

【核心功能】
• 智能错题本：自动记录错题，按章节分类整理，针对性复习薄弱环节
• 收藏管理：一键收藏重点难题，随时回顾巩固
• 多种练习模式：支持顺序练习、随机练习、专项分类练习，满足不同备考阶段需求
• 答题解析：每道题配备详细解析，答完立即查看，加深理解
• 答题卡导航：直观展示答题进度，支持快速跳转和重做
• 进度保存：自动保存练习进度，中断后可无缝续练
• 学习记录：完整记录每次练习数据，随时查看备考轨迹

【使用场景】
无论是在图书馆、自习室，还是通勤路上，打开猴哥星球就能随时进入学习状态。简洁清晰的界面设计，流畅的练习体验，让你专注于学习本身。
```

- 前 3 行最重要（在 App Store 中默认折叠，前 3 行是预览）
- 使用项目符号（• 或 -）让内容更易读
- 突出核心卖点和差异化功能
- 避免过度夸张的营销用语（如「最好」「第一」）
- 可以包含网址，但不要放价格信息（因为不同地区价格不同）

## 十四、App Store Version Release（版本发布方式）

**是否必填：** 是（默认即可）

**三个选项：**

1. **Manually release this version（手动发布）** ⭐ 推荐
   - 审核通过后不会立即上架
   - 你可以手动选择发布时间
   - 适合首次发布，可以配合营销活动安排上架时间

2. **Automatically release this version（自动发布）**
   - 审核通过后立即自动上架
   - 适合紧急修复的更新版本

3. **Automatically release this version after App Review, no earlier than（定时发布）**
   - 审核通过后在指定时间自动上架
   - 适合配合特定日期（如开学季）发布

**首次发布建议选择「手动发布」**，这样审核通过后可以选一个合适的时间点手动上架。

## 十七、常见问题

### Q1: 首次提交审核需要多久？
A: 通常 1-3 个工作日，首次审核可能稍长（3-5 天）。

### Q2: 审核被拒常见原因？
A: 本应用需注意以下几点：
- 测试账号必须可用（刷题功能需要登录）
- 隐私政策必须完整
- 应用内不能有崩溃或明显 bug
- 如果是考研题库，确保有版权或授权（如使用公开真题一般没问题）

### Q3: 截图可以用模拟器截吗？
A: 可以。使用 Xcode 模拟器截图完全符合要求，且状态栏更干净。

### Q4: 支持网址可以用 GitHub Pages 吗？
A: 可以，但建议使用自己的域名（如 www.monkeysxu.top），看起来更专业。

### Q5: 需要准备 iPad 截图吗？
A: 如果应用支持 iPad 运行，则需要。如果只在 iPhone 上运行，可以在 Xcode 中取消 iPad 支持，就不需要 iPad 截图了。
