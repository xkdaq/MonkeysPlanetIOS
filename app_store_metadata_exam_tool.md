## 四、Description（应用描述）

**是否必填：** 是

**限制：** 最多4000个字符

**建议填写：**
```
猴哥星球是一款面向考研学子的真题练习与学习辅助工具，帮助考生通过历年公开考试资料进行自测和复习，查漏补缺，提升备考效率。

【核心功能】
• 真题练习：基于公开的历年考试资料，提供顺序练习、随机练习、分类专项练习等多种模式
• 智能错题本：自动记录练习中的错题，按知识点分类整理，方便针对性复习
• 收藏管理：一键收藏重点题目，随时回顾巩固薄弱环节
• 答题解析：每道练习题目配有详细解析，帮助理解知识点和解题思路
• 答题卡导航：直观展示答题进度，支持快速跳转和错题重做
• 进度追踪：自动保存练习进度，记录每次学习数据，随时查看备考轨迹

【内容说明】
本应用提供的练习内容来源于公开渠道的历年考试资料，仅供用户个人学习、自测和复习使用。应用本身不出版、不发行任何书刊或电子出版物，所有内容以交互式练习形式呈现，而非可阅读的书刊或杂志格式。

【使用场景】
无论是在图书馆、自习室还是通勤路上，打开猴哥星球即可随时进行真题自测和错题复习。简洁的界面设计，流畅的练习体验，让你专注于学习本身。

【联系我们】
如有任何问题或建议，欢迎通过应用内「问题反馈」功能与我们联系。

官方网站：https://www.monkeysxu.top
```

**写作要点：**
- 第一句就点明"工具""自测""辅助"定位
- 用"公开的历年考试资料"代替"真题"（更中性、更官方）
- 专门增加【内容说明】段落，主动声明"不出版、不发行任何书刊"
- 强调"交互式练习形式"而非"阅读格式"
- 避免：海量、题库、刷题神器、教材、出版物、书籍、杂志等词

## 十三、风险提示与备选方案

### 风险

即使按照上述元数据修改，仍有以下风险：
1. 审核员打开App后看到"真题"内容，仍然判定为"书刊类出版物"
2. 苹果对"真题"本身的判定就是"出版物"，不论如何描述

### 如果第三次被拒

建议彻底转型为**纯工具**（Plan B）：
- 后端完全移除预置真题数据
- App变为用户自行导入/手动添加题目的空白工具
- 仅保留错题本、收藏、进度追踪等个人管理功能
- 元数据完全改为"个人错题管理工具""学习笔记工具"
- 此时App完全不涉及任何内容分发，可100%绕过出版许可问题
